# Components

Este proyecto es HTML estático con Tailwind CSS. No hay componentes reutilizables en un directorio de componentes - cada página es un archivo HTML independiente.

Los elementos UI comunes están embebidos en cada archivo HTML individual:
- Tailwind CSS via CDN
- Google Fonts (Inter)
- Material Symbols Outlined
- Firebase SDK (auth, firestore)

No existen componentes compartilhidos separados - cada página es autosuficiente.
