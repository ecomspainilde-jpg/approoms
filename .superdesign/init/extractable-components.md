# Extractable Components

Este proyecto es HTML estático. No hay componentes React/Vue/etc. para extraer.

## Elementos UI comunes (en HTML embebido)

### Layout Components
- **GlassNav** - Navegación fija con blur (embebido en cada página)
- **Footer** - Pie de página (embebido en cada página)
- **AuthModal** - Modal de login/register (embebido en landing)

### Basic Components
- **PrimaryButton** - Botones con color primary y amber-glow
- **Input** - Campos de formulario con estilos dark
- **Card** - Tarjetas con gradient-border
- **ArticleCard** - Cards para artículos del blog

Todos estos están directamente en los archivos HTML, no como componentes separados.
